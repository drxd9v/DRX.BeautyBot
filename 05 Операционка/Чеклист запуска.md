# Чеклист запуска

> Навигация: [[Индекс Obsidian]] · [[Сводная база знаний]]
>
> Связанные документы: [[Путь клиента]] · [[Сценарий продаж владельца]] · [[Дорожная карта 30-60-90]]


## Цель
Этот чеклист нужен для запуска нового клиента после оплаты.

Он помогает:
- не забыть важные шаги;
- стандартизировать запуск;
- сократить хаос;
- сделать передачу клиента в работу предсказуемой.

---

## 1. Подтверждение Оплаты
- [ ] Оплата получена
- [ ] Зафиксирован тариф
- [ ] Зафиксирован источник лида
- [ ] Если был менеджер — отмечен менеджер
- [ ] Клиенту отправлено подтверждение

---

## 2. Базовая Квалификация Клиента
- [ ] Один мастер или команда
- [ ] Ниша клиента понятна
- [ ] Понятен формат запуска
- [ ] Понятно, нужен ли базовый запуск или более полный
- [ ] Понятно, нужны ли доп. услуги

---

## 3. Подготовка Рабочего Контура
- [ ] Создан отдельный бот клиента
- [ ] Подключён токен
- [ ] Проверен owner/admin доступ
- [ ] Подключена база
- [ ] Проверены базовые переменные окружения

---

## 4. Структура Бота
- [ ] Выбран режим `solo` или `team`
- [ ] Если `solo` — назначен основной мастер
- [ ] Если `team` — добавлены мастера
- [ ] Проверена логика выбора мастера

---

## 5. Контент
- [ ] Добавлены услуги
- [ ] Проверены цены
- [ ] Добавлены категории портфолио
- [ ] Загружены фото
- [ ] Проверены названия и тексты

---

## 6. Расписание
- [ ] Настроены слоты
- [ ] Проверены закрытые даты
- [ ] Проверена логика свободных окон
- [ ] Проверено отображение календаря

---

## 7. Клиентский Путь
- [ ] Проверена запись
- [ ] Проверен прайс
- [ ] Проверено портфолио
- [ ] Проверены кнопки назад
- [ ] Проверен шаг контакта
- [ ] Проверено подтверждение записи

---

## 8. Уведомления
- [ ] Проверено уведомление мастеру о новой записи
- [ ] Проверено уведомление клиенту
- [ ] Проверены напоминания
- [ ] Проверена логика за 1 час / 2 часа, если используется

---

## 9. Админка
- [ ] Проверены записи на сегодня
- [ ] Проверены будущие записи
- [ ] Проверено управление услугами
- [ ] Проверено управление портфолио
- [ ] Проверено управление мастерами
- [ ] Проверена отмена записи

---

## 10. Продакшн-Проверка
- [ ] Бот запускается без ошибок
- [ ] Нет критичных UX-багов
- [ ] Нет битой кодировки
- [ ] Нет сломанных кнопок
- [ ] Нет конфликта ролей

---

## 11. Передача Клиенту
- [ ] Клиенту отправлен доступ
- [ ] Клиент понимает, как пользоваться ботом
- [ ] Клиент знает, как добавлять услуги и портфолио
- [ ] Клиент понимает, как смотреть записи
- [ ] Клиент знает следующий шаг после запуска

---

## 12. После Запуска
- [ ] Проверить, что клиент зашёл
- [ ] Проверить, что клиент понимает базовый сценарий
- [ ] Уточнить, нужен ли дополнительный вопрос/помощь
- [ ] Зафиксировать запуск как завершённый

---

## Короткая Формула
Оплата -> квалификация -> настройка -> проверка -> передача -> запуск.
